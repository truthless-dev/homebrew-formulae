# Truthless-dev Formulae

## How do I install these formulae?

`brew install truthless-dev/formulae/<formula>`

Or `brew tap truthless-dev/formulae` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "truthless-dev/formulae"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
